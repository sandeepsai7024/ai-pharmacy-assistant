# 💊 AI Pharmacy Assistant

A simple AI-powered tool that helps users quickly understand medicines, their uses, and basic alternatives.

---

## 🚀 Why I built this

Coming from a pharmacy background, I noticed that accessing quick and clear medicine information can be time-consuming, especially in busy environments.

So I built a lightweight AI assistant that simplifies this process.

---

## 🧠 What it does

- Takes a medicine name as input  
- Provides:
  - Usage information  
  - Category  
  - Basic alternatives  
- Generates responses instantly using AI  

---

## 🛠️ Tech Used

- Python  
- OpenAI API (or ChatGPT logic)  
- Basic UI (Replit / Streamlit / simple web app)
